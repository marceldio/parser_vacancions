# parser_vacancions
Project for collecting vacancies

Программа, которая будет получать информацию о вакансиях с платформы hh.ru в России, сохранять ее в файл и позволять удобно работать с ней: добавлять, фильтровать, удалять.

Документация для сбора вакансий с hh.ru
Ссылка на API: https://github.com/hhru/api/.

Выходные данные

    Информация о вакансиях, полученная с разных платформ,
    сохраненная в JSON-файл.
    Отфильтрованные и отсортированные вакансии, 
    выводимые пользователю через консоль.
